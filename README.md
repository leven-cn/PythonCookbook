# PythonStart

## Why Python?

### 市场需求

**人工智能时代**

BOSS直聘研究院数据显示，由于大批公司涉足**AI领域**，
2017年AI人才需求已达到2016年的两倍，2015年的5.3倍，人才需求直线上升，
大部分岗位三年复合增长率超过`200%`。AI应用层职位需求增速尤为显著，
增速最高的三个岗位依次是**算法工程师**、**语音识别**和**图像处理**。
