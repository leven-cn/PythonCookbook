

## 基础类型

- `type()`
- `isinstance()`
- `bool`, `int`, `float`, `str`, `None`

### 布尔类型(`bool`)

- `True` / `False`
- `not`, `and`, `or`
- `is`
- 断言(**assertion**): `assert`, `AssertionError`

### 整数型(`int`)

- `0b`/`0o`/`0x`
- `bin()`, `oct()`, `hex()`

### 浮点数(`float`)

- `float('inf')`, `float('-inf')`, `float('nan')`
- `round()`

## 操作符(operator)

- `+`, `-`, `*`, `/`, `//`, `%`
- `()`
- `==`, `!=`, `<`, `>`, `>=`, `<=`
